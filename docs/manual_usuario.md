# Manual rapido

## 1. Acessar
- Execute a aplicacao e abra `http://127.0.0.1:5000`.
- Entre com usuario admin inicial.

## 2. Dashboard
- Visualize total de produtos e itens abaixo do minimo.
- Use os atalhos para listar estoque e, se admin, cadastrar produto/usuario.

## 3. Produtos
- Admin pode cadastrar e editar produtos.
- Na listagem, itens abaixo do minimo aparecem com destaque.
- Admin pode registrar entrada/saida direto na tabela.

## 4. Usuarios
- Apenas admin acessa tela de usuarios.
- Apenas admin pode cadastrar novo usuario.

## 5. Seguranca
- O sistema exige login.
- Sessao e encerrada no logout.
- Senhas sao armazenadas como hash.
